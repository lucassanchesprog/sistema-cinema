# Sistema de Cinema (Python)

Projeto de simulação de compra de ingressos em um cinema.

## Funcionalidades
- Mostrar filmes disponíveis
- Escolher um filme pelo número
- Mensagem de confirmação da escolha

## Como rodar
1. Abra o terminal
2. Execute: `python main.py`
